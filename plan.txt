هيكل المنصة العام (Architecture)

عند الدخول إلى النظام: - تسجيل دخول - لوحة تحكم رئيسية - قائمة جانبية
للتنقل بين الأقسام

التقسيم الأساسي: - Dashboard - POS (الكاشير) - الكتب - المخزون -
المبيعات - المشتريات (اختياري) - التقارير - المستخدمون والصلاحيات -
الإعدادات

=====================================

لوحة التحكم (Dashboard) - مبيعات اليوم - عدد الفواتير - عدد الكتب -
الكتب منخفضة المخزون - رسم بياني للمبيعات صفحة للعرض فقط.

=====================================

الكاشير (POS) - مسح الباركود - بحث سريع بالاسم - قائمة المنتجات في
الفاتورة - تعديل الكمية / حذف - الإجمالي والخصم - طباعة الفاتورة - إتمام
البيع

متطلبات التصميم: - أزرار كبيرة - سرعة عالية - واجهة بسيطة

=====================================

إدارة الكتب الجدول: - اسم الكتاب - المؤلف - السعر - الباركود - الكمية -
تعديل / حذف

إضافة كتاب جديد: - الاسم - التصنيف - السعر - الباركود - صورة (اختياري) -
كمية ابتدائية

=====================================

المخزون - زيادة كمية - خصم كمية - جرد المخزون - سجل الحركات (Stock
History)

=====================================

المبيعات - رقم الفاتورة - التاريخ - الموظف - الإجمالي - تفاصيل
الفاتورة - طباعة PDF

=====================================

التقارير - يومية - شهرية - سنوية - أكثر الكتب مبيعًا - الأرباح - المخزون
الناقص - تصدير Excel / PDF

=====================================

المستخدمون والصلاحيات - إضافة موظف - تحديد الدور: مدير كاشير مخزن

=====================================

الإعدادات - اسم المكتبة - الشعار - الضريبة - العملة - الطابعة - النسخ
الاحتياطي

=====================================

التقنيات التي ستنفذ بها: Backend: Django Database: PostgreSQL Frontend: HTML +
CSS + Bootstrap + JavaScript Barcode Scanner PDF Reports
